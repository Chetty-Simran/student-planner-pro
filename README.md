#  Student Planner Pro details

A full-stack ASP.NET Core MVC web application designed to help university students organise their academic life. Student Planner Pro enables users to manage assignments, monitor deadlines, track progress, and improve productivity through an intuitive and modern interface.

---

## Overview

Student Planner Pro was developed as a portfolio project while studying towards a Bachelor of Computer and Information Sciences (Application Development).

The aim of the project is to demonstrate software engineering principles, database design, and full-stack web development using Microsoft's .NET ecosystem.

---

## Current Features

- Create assignments
- View all assignments
- Edit assignments
- Delete assignments
- SQL Server database integration
- Entity Framework Core
- MVC Architecture
- Git version control
- Priority levels
- Assignment status tracking
- Assignment descriptions
- Data validation
